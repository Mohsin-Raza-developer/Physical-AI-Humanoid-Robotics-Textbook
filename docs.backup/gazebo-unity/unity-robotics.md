# Unity Robotics Simulation

Unity provides a powerful real-time 3D development platform that can be used for robotics simulation with the Unity Robotics packages.

## Key Features

- High-fidelity graphics and rendering
- Physics simulation with NVIDIA PhysX
- ROS# bridge for ROS communication
- Synthetic data generation
